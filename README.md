# GATE Simulator
**By:** A. Hardy

### Instructor Thoughts
Your classmates really seemed to enjoy the game, so that’s a plus. Also grateful to be the only instructor that didn’t have a silly picture of their face in the game. :) 
The Astrophysics section was unbelievably touchy and such a long animation to go through when you know you didn’t get it right. After retrying an uncountable number of times, we did finally finish the game. Honestly, we were a little disappointed you didn’t get the experience of using Unity’s full-featured 3D engine, but won’t fault you for wanting to use Godot, especially if you already have some experience with it.


### Tweaks we made post-class
We were able to get the WebGL build to load, but it was running extremely choppy, so we decided not to post that publicly, but the standard game is available for download.

### Moving Forward
If you are interested in making a more complete 3D game in Godot and aren’t already familiar with what you would need to do so, there are tutorials referenced on our website you might appreciate that are specifically about building 3D games in Godot.
